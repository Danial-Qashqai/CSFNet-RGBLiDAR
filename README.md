## Overview

This repository presents **CSFNet-RGBLiDAR**, a real-time RGB-LiDAR semantic segmentation model trained using projected LiDAR data on a subset of the KITTI-360 dataset.

The proposed model is based on **CSFNet**, our previously introduced semantic segmentation framework. The original CSFNet architecture and implementation are available at:

[CSFNet: https://github.com/Danial-Qashqai/CSFNet](https://github.com/Danial-Qashqai/CSFNet)

In this work, CSFNet is extended to the RGB-LiDAR domain by incorporating projected LiDAR information for multi-modal semantic segmentation.
